Online Job Portal

About
This is a React-based online job portal that allows users to search and apply for jobs. The portal uses Firebase as its backend database.

Features
Search for jobs by title, location, experience, and type
Filter search results by job criteria
Display job details, including title, company, type, experience, and location
Apply for jobs directly from the portal

Components
Navbar: a navigation bar that appears at the top of the page
Header: a header component that displays the portal's title and tagline
SearchBar: a search bar that allows users to search for jobs by criteria
JobCard: a job card component that displays job details

Firebase Configuration
Firebase is used as the backend database for the portal
The Firebase configuration is stored in firebase.config.js
The Firebase database is initialized in firebase.config.js and exported as db

Getting Started
Clone the repository and run npm install to install dependencies
Run npm start to start the development server
Open http://localhost:3000 in your browser to access the portal
